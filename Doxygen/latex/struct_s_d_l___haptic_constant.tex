\doxysubsection{Detailed Description}
A structure containing a template for a Constant effect. 

This struct is exclusively for the \mbox{\hyperlink{_s_d_l__haptic_8h_a955fb9f680dcf9cc72a3d5263e85b80a}{SDL\+\_\+\+HAPTIC\+\_\+\+CONSTANT}} effect.

A constant effect applies a constant force in the specified direction to the joystick.
