Check if SDL\+\_\+ttf is initialized.

This reports the number of times the library has been initialized by a call to \mbox{\hyperlink{_s_d_l__ttf_8h_a766731737c2ca9121924bcef57889128}{TTF\+\_\+\+Init()}}, without a paired deinitialization request from \mbox{\hyperlink{_s_d_l__ttf_8h_aa215d7d4fe202960904507a6ad4e6097}{TTF\+\_\+\+Quit()}}.

In short\+: if it\textquotesingle{}s greater than zero, the library is currently initialized and ready to work. If zero, it is not initialized.

Despite the return value being a signed integer, this function should not return a negative number.

\begin{DoxyReturn}{Returns}
the current number of initialization calls, that need to eventually be paired with this many calls to \mbox{\hyperlink{_s_d_l__ttf_8h_aa215d7d4fe202960904507a6ad4e6097}{TTF\+\_\+\+Quit()}}.
\end{DoxyReturn}
